# EmojiPrediction

## Requirements
python3

tensorflow >= 1.9.0

## Usage
You need to download the pretrained embeddings and put it under [data/word2vec] dir first.

english preptrained embedding:
- https://github.com/fvancesco/acmmm2016

datasets:
- train set: 466,233
- valid set: 50,000
- test  set: 50,000
```
python main.py
```
